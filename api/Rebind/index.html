Rebind

A Rebind is used for getting an input the user actuates.

You can create a query with specific devices and inputs and then call Rebind:start to get a promise that resolves with the first input matching the query that the user actuates. 

local input = Rebind.new(ValueKind.Boolean)
	:withDevices({ Devices.Keyboard })
	:withoutInputs({ Devices.Keyboard.Space })
	:start()
	:expect()
